# Toy IR/Raman Prototype (Archived)

This directory contains an earlier toy IR/Raman prototype and is not part of the final DP Technology submission demo.

The final reviewer-facing demo is under `03_code/`, `04_data/`, and `05_outputs/`.

## Contents

This archive preserves:
- `configs/` – Old YAML demo configuration
- `data/` – Toy synthetic sample spectra (12 molecules)
- `scripts/` – Old CLI entry point (`run_demo.py`, `validate_outputs.py`)
- `src/multispec_geodiff/` – Old Python package (data_io, geometry, molecular_filters, etc.)
- `tests/` – Old test suite (8 test files, 17 tests)
- `outputs/` – Old output files (spectrum figures, CSVs, trace logs)
- `docs/` – Old documentation files
- `notebooks/` – Old Jupyter notebook
- `assets/` – Empty placeholder directory
- `latex/` – Old LaTeX submission files
- Root-level design/planning files (AGENTS.md, Design.md, etc.)
